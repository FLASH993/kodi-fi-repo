Welcome to the Kodi-Fi Repo

This repository contains the official Kodi-Fi build and OpenWizard installer.

Installation Instructions

To install this repository in Kodi:

  1. Go to Settings > File Manager > Add Source
  2. Enter this URL: https://flash993.github.io/kodi-fi-repo/repo/repository.kodifi/
  3. Go to Add-ons > Install from zip file and choose the source
  4. Install the Kodi-Fi Repository
  5. Then install OpenWizard from the repo and apply your build
